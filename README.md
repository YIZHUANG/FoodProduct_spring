
Build with Java , reactJs and Postgresql and mongodb.

Add your own postgresql / MariaDB / mySql database in the application.properties files.


## Install project dependencies
```
$ npm install
```

## Run webpack
```
webpack
```

## Run the app:

mvnw spring-boot:run
